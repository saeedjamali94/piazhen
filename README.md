## 🛠 Technology Stack

- **Framework:** Wordpress +6
- **PHP Version:** 8.1+
- **Database:** MySQL


## 📦 Prerequisites

- Wordpress +6 Version
- PHP 8.1 or higher
- Node.js and NPM (for frontend assets)
- MySQL database
- XAMPP (Windows) or equivalent local server
